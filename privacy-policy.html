Kebijakan Privasi

Berlaku sejak: Mei 2025 Diperbarui: Mei 2026
PENDAHULUAN
Tentang Kebijakan Ini

Yayasan Hidayatul Amin Sagatama ("kami") berkomitmen untuk melindungi privasi dan keamanan data pribadi pengguna yang mengakses layanan pembayaran Pi Network kami di https://website-sagatama.vercel.app.

Kebijakan ini menjelaskan data apa yang kami kumpulkan, bagaimana kami menggunakannya, dan hak-hak Anda sebagai pengguna.

PASAL 1
Data yang Kami Kumpulkan

Saat Anda menggunakan layanan kami melalui Pi Browser, kami mengumpulkan data berikut:

  • Username Pi Network Anda (diperoleh melalui Pi SDK dengan izin Anda)
  • Wallet address Pi Network Anda (untuk keperluan transaksi)
  • Informasi pembayaran: jumlah Pi, jenis transaksi, dan memo
  • Data formulir yang Anda isi sendiri: nama, nomor WhatsApp, alamat pengiriman, dan pesan
  • Waktu dan tanggal transaksi
PASAL 2
Bagaimana Kami Menggunakan Data

Data yang kami kumpulkan digunakan semata-mata untuk:

  • Memproses dan mengkonfirmasi pembayaran Pi Network Anda
  • Mengirimkan produk atau layanan yang Anda pesan
  • Mencatat riwayat donasi dan transaksi secara transparan
  • Menghubungi Anda terkait status pesanan jika diperlukan
  • Mematuhi kewajiban hukum dan audit internal yayasan

Kami tidak menjual, menyewakan, atau membagikan data pribadi Anda kepada pihak ketiga untuk tujuan komersial.

PASAL 3
Integrasi Pi Network

Layanan ini menggunakan Pi Network SDK untuk autentikasi dan pembayaran. Dengan mengklik "Hubungkan Pi Account", Anda memberikan izin kepada aplikasi ini untuk mengakses:

  • Username Pi Network (untuk identifikasi)
  • Akses pembayaran (untuk memproses transaksi)
  • Wallet address (untuk verifikasi transaksi)

Proses autentikasi sepenuhnya dikelola oleh Pi Network. Kami tidak menyimpan password atau kunci privat wallet Anda. Untuk informasi lebih lanjut tentang privasi Pi Network, kunjungi kebijakan resmi Pi Network.

PASAL 4
Penyimpanan & Keamanan Data

Data transaksi disimpan di Google Firebase Firestore dengan enkripsi standar industri. Kami menerapkan langkah-langkah keamanan teknis dan organisasi yang wajar untuk melindungi data Anda dari akses tidak sah, kehilangan, atau pengungkapan.

  • Data sesi login disimpan sementara di browser (sessionStorage) dan dihapus saat sesi berakhir
  • Riwayat transaksi disimpan di server Firebase dengan akses terbatas
  • Tidak ada data kartu kredit atau informasi pembayaran konvensional yang disimpan
PASAL 5
Hak-Hak Anda

Sebagai pengguna, Anda berhak untuk:

  • Meminta akses ke data pribadi Anda yang kami simpan
  • Meminta koreksi data yang tidak akurat
  • Meminta penghapusan data Anda (dengan batasan kewajiban hukum)
  • Mencabut izin akses Pi Network kapan saja melalui aplikasi Pi

Untuk menggunakan hak-hak di atas, hubungi kami melalui informasi kontak di bawah.

PASAL 6
Perubahan Kebijakan

Kami dapat memperbarui kebijakan privasi ini sewaktu-waktu. Perubahan akan diberitahukan melalui halaman ini dengan memperbarui tanggal "Diperbarui" di bagian atas. Penggunaan layanan kami setelah perubahan diterbitkan berarti Anda menyetujui kebijakan yang diperbarui.
